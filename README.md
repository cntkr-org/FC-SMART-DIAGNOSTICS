# FC Smart Diagnostics - Professional Clinic Website

A clean, professional, and responsive website for your clinic featuring appointment booking, service listings, pricing information, and contact details.

## Files Included

- **index.html** - Main website file with all sections
- **styles.css** - Complete styling and responsive design
- **script.js** - Interactive features and form handling

## Features

✅ Responsive design (works on desktop, tablet, mobile)
✅ Professional gradient hero section
✅ Clinic about section
✅ Services listing
✅ Service pricing
✅ Appointment booking form
✅ Contact information
✅ WhatsApp contact link
✅ Smooth scrolling navigation
✅ Hover animations

## How to Use

### 1. **Quick Start**
   - Open `index.html` in your web browser
   - The website will display immediately

### 2. **Customize Your Information**

Edit the `index.html` file and replace these placeholders:

**Clinic Name:**
```html
<div class="logo">FC Smart Diagnostics</div>
```

**Location Address:**
```html
<p>123 Medical Plaza, Healthcare Street</p>
<p>City, State - 12345</p>
```

**Contact Details:**
```html
<p>Phone: +91-XXX-XXX-XXXX</p>
<p>Email: info@fcsmartdiagnostics.com</p>
<a href="https://wa.me/919999999999">WhatsApp</a>
```

Update the WhatsApp link with your number:
- Replace `919999999999` with your WhatsApp number (country code + number, no spaces or hyphens)

**Business Hours:**
```html
<p>Monday - Friday: 9:00 AM - 6:00 PM</p>
<p>Saturday: 9:00 AM - 2:00 PM</p>
```

**Services:**
Update the service cards in the Services section with your actual services.

**Pricing:**
Update the pricing section with your actual costs.

### 3. **Form Submission**

Currently, the booking form shows a confirmation message. To send form submissions to your email:

Option A: **Use a free form service (easiest)**
1. Go to formspree.io or getform.io
2. Create an account and set up your form
3. Replace the form ID in `script.js`

Option B: **Use backend/hosting with email**
Contact your hosting provider about email setup

## Color Customization

Edit these colors in `styles.css` (line ~9-12):
```css
--primary-color: #0066cc;      /* Blue - main color */
--secondary-color: #00a86b;    /* Green - accent color */
--dark-color: #1a1a1a;         /* Dark - navbar background */
```
